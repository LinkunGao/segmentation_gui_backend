# FastApi backend

### Dependencies
- pip install "fastapi[all]"

### Start serve
```bash
uvicorn main:app --reload
```

### Docker

- start
```bash
docker-compose up
```
- shutdown
```bash
docker-compose down
```

## Set up SPARC dataset

- copy all contents of the SPARC dataset rather than the dataset folder itself.
- paste all contents in `./data` folder.
